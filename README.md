# 💼 Payroll Management System

A Python-based Payroll Management System to manage employee records, salary details, and payroll operations efficiently.

---

## 🚀 Features
- Add and manage employee records  
- Store salary details  
- Calculate and track payroll  
- Simple and user-friendly interface  

---

## 🛠️ Tech Stack
- Python  
- Tkinter (if GUI used)  
- File Handling (PKL files)  

---

## 📂 Project Structure
- `payroll.py` → Main application file  
- `employees.pkl` → Stores employee data  
- `salaries.pkl` → Stores salary data  

---

## ▶️ How to Run
1. Install Python  
2. Download/Clone this repository  
3. Run the file:
   ```bash
   python payroll.py
